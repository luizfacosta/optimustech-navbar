Projeto desenvolvido como parte do desafio #7DaysOfCode da Alura, focado no desenvolvimento de uma interface moderna e funcional.

💻 Tecnologias
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização avançada com o uso de variáveis nativas (:root) para padronização de cores.

Google Fonts: Implementação da fonte Inter para uma tipografia limpa.

🎯 Objetivo
Construir a primeira seção e o cabeçalho de uma página de recrutamento, seguindo fielmente o layout proposto no Figma.

📸 Preview
(Dica: Se você salvar um print da tela como preview.png na pasta do seu projeto, essa imagem aparecerá aqui).

🌐 Deploy
Confira o projeto online: https://luizfacosta.github.io/optimustech-navbar/.

📚 Aprendizados do Dia 2
Variáveis CSS: Centralização da paleta de cores para facilitar a manutenção do código.

Hierarquia Visual: Criação da seção "Sobre nós" com foco em alinhamento e contraste tipográfico.

Responsividade: Noções de containers com larguras máximas (max-width) para melhor leitura em diferentes telas.
